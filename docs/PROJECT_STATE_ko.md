# 프로젝트 현황 요약: 타워 디펜스

## 개요
- 단일 HTML5 캔버스 기반 타워 디펜스 (index.html, style.css, main.js)
- 좌측 접이식 빌드 패널·중앙 전장·우측 정보 칼럼을 갖춘 글래스모피즘 UI
- 설치/업그레이드/격파/폭발/레이저/패널 토글에 맞춘 Web Audio 사운드와 HUD 음소거 토글 제공

## 파일 구성
- index.html – 상단 HUD와 사운드 토글, 좌측 빌드 패널(#build-panel), 전장 캔버스, 웨이브/포탑/적 정보 카드, 패배 오버레이
- style.css – 배경 그라데이션, 글래스 패널, 포탑 카드 그리드, 빌드 패널 반응형 접기 로직, 버튼/타이포그래피 개선
- main.js – 포탑 정의, 빌드 패널 동적 생성, 게임 상태/로직, 오디오 엔진, 공격 패턴(샷건·레이저·광역·박격포), 렌더링 및 이벤트 바인딩

## 조작 & UI
- 빌드 패널(좌측)에서 포탑 카드를 선택 후 전장을 좌클릭하면 설치
- 포탑 우클릭으로 업그레이드(최대 15레벨, 레벨당 피해 2.5배)
- 우측 정보 칼럼은 선택한 포탑/적 정보를 즉시 반영, 웨이브 미리보기에는 상태·남은 적·체력·속도·보상이 표시
- 상단 HUD: 골드 입력(+100/+500), 웨이브 이동, 속도(1x/2x/3x), 사운드 토글(🔊/🔇)
- 스페이스바로 일시 정지/재개, 패배 시 오버레이에서 재시작 또는 닫기 선택

## 적 웨이브
- 웨이브당 적 수: 8 + floor(wave * 1.5)
- 체력 증폭 ENEMY_HP_GROWTH_RATE = 1.25, 이동 속도(49)와 보상(14)은 고정
- 웨이브 미리보기: 상태(대기/휴식/전투 중/패배), 남은 적, 체력, 타일/초 속도, 획득 골드

## 포탑 & 업그레이드
- TOWER_TYPES에 정의된 8종 포탑
  - **기본**: 균형 잡힌 단일 탄환
  - **샷건**: 근거리 산탄 피해
  - **저격**: 장거리 고속 빔 탄환
  - **점사**: 짧은 지연을 둔 소형 연사
  - **연사**: 높은 발사 속도
  - **폭발**: 명중/만료 시 광역 폭발
  - **레이저**: 지속적으로 타격하는 빔 피해
  - **박격포**: 포물선 궤적 + 광역 대미지
- 업그레이드 시 골드를 소비하여 레벨 +1, 피해 ×2.5, 최대 레벨 15, 타입별 기본값에서 업그레이드 비용은 매 레벨 2배

## 사운드
- 첫 사용자 상호작용 시 Web Audio Context를 초기화하고 마스터 게인으로 믹싱
- 제공 사운드: select, 	oggle, uild, upgrade, kill, explosion, laser
- HUD 사운드 토글이 아이콘/aria 상태를 갱신하고 마스터 게인을 완전히 음소거/해제

## 전투 로직 핵심
- performTowerAttack()이 패턴별(샷건, 점사, 폭발, 레이저, 박격포) 투사체/빔을 생성
- handleLaserAttack()은 지속 피해·빔 시각효과를 담당하며 최초 타격 시 사운드 재생
- pplyExplosion()이 광역 피해, 이펙트, 폭발 사운드를 처리
- update()가 스폰·타깃팅·투사체 이동·광역 처리·골드/라이프 갱신·UI 반영을 수행

## 렌더링
- 캔버스 렌더 순서: 경로/그리드/호버 → 포탑(타입별 도형) → 적 HP 바 → 투사체(빔/삼각/육각/구/궤적) → 충격 파동 → 활성 레이저 → 일시 정지/패배 오버레이
- 빌드 패널 및 카드 스타일은 CSS 유틸리티 기반, 포탑 선택 시 사거리 링 표시

## 참고 사항
- UI 문자열은 모두 한글이므로 UTF-8 인코딩을 유지해야 글자가 깨지지 않음
- 자동화 테스트 없음: 포탑 설치/업그레이드(15레벨), 광역 상호작용, 레이저 효과, 사운드 토글, 반응형 레이아웃 등을 수동 점검 권장
- UI나 메커닉 변경 시 docs/PROJECT_STATE.md와 docs/PROJECT_STATE_ko.md를 함께 갱신할 것
