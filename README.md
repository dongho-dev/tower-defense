# 타워 디펜스 커맨드 (Tower Defense Command)

한 화면에서 빠르게 즐기는 HTML5 타워 디펜스 게임입니다. 순차적으로 등장하는 적 웨이브를 막기 위해 다양한 포탑을 배치하고 업그레이드하세요. 최근 UI 리뉴얼과 Web Audio 효과음이 더해져 데스크톱 브라우저에서 바로 플레이하기 좋습니다.

## 주요 특징
- **8종 포탑**: 기본/샷건/저격/점사/연사/폭발/레이저/박격포. 각기 다른 사거리·피해·효과(산탄, 광역, 지속 빔 등)를 보유합니다.
- **15레벨 업그레이드**: 우클릭으로 업그레이드하며 레벨마다 피해가 2.5배씩 증가합니다.
- **직관적 HUD**: 좌측 빌드 패널, 상단 골드·웨이브 조절, 우측 정보 카드, 사운드 토글 제공.
- **Web Audio 효과음**: 포탑 선택/설치/업그레이드/적 처치/폭발/레이저 등에 맞춰 사운드를 재생합니다.
- **순차 웨이브**: 웨이브 종료 후 짧은 휴식이 주어지며, 웨이브 미리보기 카드에서 예상 정보를 확인할 수 있습니다.
- **간단 테스트**: jsdom 기반 스모크 테스트로 기본 UI 요소를 빠르게 검증합니다.

## 데모 실행
1. 저장소를 클론하거나 ZIP으로 내려받습니다.
2. `index.html`을 데스크톱 브라우저(Chrome/Edge/Firefox 등)에서 직접 열면 즉시 플레이 가능합니다.

> 캔버스 렌더링이 제대로 표시되지 않을 경우 새로고침(Ctrl+F5) 후 다시 시도해주세요.

## 개발 환경
```
# 의존성 설치 (테스트용)
npm install

# 스모크 테스트 실행
npm test
```

테스트는 jsdom으로 DOM을 흉내 내어 빌드 패널, 속도 버튼, 사운드 토글 등이 정상 생성되는지 확인합니다.

## 플레이 방법
- **포탑 설치**: 좌측 빌드 패널에서 포탑 카드 선택 → 전장 캔버스를 좌클릭
- **업그레이드**: 설치된 포탑 우클릭 (최대 15레벨)
- **HUD 제어**
  - 골드 입력 및 +100/+500 버튼으로 자금 조절
  - 웨이브 입력/이동으로 원하는 웨이브 바로 호출
  - 속도 버튼(1x/2x/3x)과 사운드 토글(🔊/🔇)
  - 스페이스바로 일시 정지/재개
- **패배 시**: 오버레이에서 다시 시작 또는 닫기 선택

## 디렉터리 구조
```
project-root/
├── index.html          # 게임 진입점 (HUD/패널/캔버스)
├── style.css           # 글래스모피즘 UI 스타일
├── main.js             # 게임 로직, 포탑/적/렌더링/오디오
├── tests/smoke.test.js # jsdom 기반 스모크 테스트
├── docs/               # 프로젝트 상태 문서 (한글/영문)
└── package.json        # 테스트 실행 스크립트 및 jsdom 의존성
```
